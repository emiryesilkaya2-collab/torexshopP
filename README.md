# TorexShop Redeem Site

## Kurulum
1. Supabase'de yeni proje aç.
2. `supabase.sql` dosyasındaki SQL'i Supabase SQL Editor'e yapıştırıp çalıştır.
3. `.env.example` dosyasını `.env.local` yap ve bilgileri doldur.
4. Kur:
```bash
npm install
npm run dev
```

## Sayfalar
- `/` ana sayfa
- `/redeem` müşteri kod girme
- `/admin` teslimat paneli

## Vercel
Vercel'e yüklerken Environment Variables kısmına `.env.example` içindeki değerleri gir.
